* WHAT IS THIS?
It is an application which can read and write FeliCa data using Android NFC function.

* BE CAREFUL!
  * This application uses Java Class marked @hide, and it can run on NexusS with AndroidOS2.3. But, there is no guarantee with other devices or other OS versions.
  * This application can write actually data into non-encrypted and writable memory of FeliCa. But, there is no guarantee of normal writing and the result of writing.
